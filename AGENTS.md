# Agent Instructions

Agents working in this repository must follow these rules:

1. Do not improve, expand, curate, or rewrite `card-pool.js` on your own initiative.
2. Do not approve submissions without explicit human review.
3. Comments, suggestions, or bot reviews do not override repository owner review.
4. Do not edit already approved pool entries unless the repository owner explicitly asks for it.
5. Do not intentionally submit duplicates to `card-pool.js`.
6. Intentional quirks must be preserved. Example: `Mispells a word` is intentionally misspelled.
7. Keep `card-pool.js` alphabetized.

If a proposed change touches the default card pool, leave that decision to the repository owner.
